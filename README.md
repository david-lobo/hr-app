
# HR APP

This is a simple HR App built with Laravel 8, React, Bootstrap 4.  

To run this app:

- `git clone` the repo
- `cd hr-app`
- `composer install`
- `php artisan migrate:fresh`
- `php artisan db:seed`
- `php artisan serve`
- `npm install`
- `npm run watch`

The app should be running locally and will be available here:

http://127.0.0.1:8000/
